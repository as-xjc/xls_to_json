xls_to_language_code
===========
  a tool for xls to language code

## depend:
  * python 3+
  * xlrd

## useage:
  * python xls_to_code.py xls_path sheet_name to_code_path language struct_row

  example:  
    python3 xls_to_code.py example/test.xls test_map ./server.json json server
